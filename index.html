소개

미래의 정보 교사

현재 컴퓨터공학과에 재학 중이며, 정보 교사가 되어 학생들에게 프로그래밍의 즐거움과 컴퓨터 사고력을 가르치는 것이 꿈입니다. 다양한 프로그래밍 언어를 학습하며 교육에 활용할 수 있는 프로젝트들을 만들어가고 있습니다.

학업 외에는 베이스와 피아노 연주를 즐기며, 규칙적인 헬스와 런닝으로 건강한 라이프스타일을 유지하고 있습니다. 음악과 운동을 통해 스트레스를 해소하고 새로운 에너지를 얻고 있어요.

기술 스킬

💻

C 언어

프로그래밍의 기초를 다진 첫 번째 언어로, 게임 개발 프로젝트들을 통해 숙련도를 높였습니다.

🌐

JavaScript

웹 개발의 핵심 언어로, 간단하게 웹 페이지를 만들어보며 점점 흥미를 쌓고 있습니다.

🐍

Python

데이터 처리와 알고리즘 구현에 활용하며, 교육용 프로그램 개발에도 관심이 많습니다.

📱

Kotlin

안드로이드 앱 개발을 위해 학습한 언어로, 모바일 애플리케이션 프로젝트에 활용했습니다.

프로젝트

숫자야구 게임

C언어로 개발한 첫 번째 프로젝트. 사용자가 숫자를 맞추는 게임으로 기본적인 프로그래밍 로직을 학습했습니다.

창세기존

C언어 기반의 게임 프로젝트. 조그마한 맵 안에서 적과 전투를 하는 프로그램을 만들어 프로그래밍에 대한 재미를 상승시켰습니다.

야옹이와 수프

C언어로 제작한 게임. 고양이의 호감도를 쌓으면 고양이가 수프를 만드는 프로그램을 개발했습니다.

찹찹페이

Kotlin으로 개발한 안드로이드 앱. 부가가치세 정산을 도와주는 실용적인 애플리케이션으로 모바일 개발 경험을 쌓았습니다.
